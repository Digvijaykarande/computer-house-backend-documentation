üõí Product APIs

GET /products

Fetch all products (supports pagination).

GET /products/{id}

Fetch product by ID.

POST /products

Add product with image (multipart/form-data).

{
  "name": "Laptop",
  "brand": "Dell",
  "category": "Electronics",
  "price": 75000,
  "available": true
}
DELETE /products/{id}

Delete product by ID.
